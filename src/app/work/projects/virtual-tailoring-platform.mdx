## Overview

Led the development of an end-to-end MVP for a virtual tailoring platform. The
system allows users to customize garments and see realistic AR previews while
receiving AI-driven style recommendations.

## Key Features

- **AI Style Recommendations**: Personalized suggestions based on user
  preferences and body type.
- **AR Try-On Experience**: Real-time garment visualization using web
  technologies.
- **Full E-commerce Flow**: From customization to order placement with backend
  integration.
- **Admin Dashboard**: For managing products, orders, and analytics.

## Technologies Used

- **Frontend**: Next.js, React, TypeScript, Tailwind
- **AR**: WebXR / Three.js
- **AI**: OpenAI for recommendations
- **Backend**: Node.js, Supabase, PostgreSQL
- **Deployment**: Vercel + AWS services

## Challenges and Learnings

Performance optimization for AR on mobile devices was the biggest hurdle.
Implemented lazy loading, model compression, and caching strategies. Also
learned how to balance AI creativity with practical fashion constraints.

## Outcome

- Delivered production-ready MVP in **4 months**
- **50% reduction** in time-to-market compared to initial estimates
- Strong user engagement with the AR experience
- Solid foundation for future iterations and investor demos
