## Overview

Developed the core backend and several frontend modules for a multi-tenant SaaS
platform that handles event registration, ticketing, attendee management, and
real-time updates.

## Key Features

- **Real-time Booking System**: With availability checking and waitlists.
- **Multi-tenant Architecture**: Isolated data per client/organizer.
- **Analytics Dashboard**: Key metrics and attendee insights.
- **Integrations**: Payment gateways, email notifications, and calendar sync.

## Technologies Used

- **Backend**: Node.js, TypeScript, PostgreSQL, AWS Lambda
- **Frontend**: Angular / React (depending on module)
- **Realtime**: Supabase Realtime
- **Infrastructure**: Docker, CI/CD pipelines

## Challenges and Learnings

Scaling for concurrent events with high traffic required careful database design
and caching strategies. Implemented robust error handling and monitoring from
day one.

## Outcome

- **40% reduction** in debugging and maintenance time
- Improved system reliability under load
- Enabled the client to manage significantly more events with the same team
- Modernized legacy parts of the platform increasing overall performance
